Graduado em Análise e Desenvolvimento de Sistemas pelo Unisenai, com formação técnica em Desenvolvimento de Sistemas. Atuei como Desenvolvedor Web Fullstack na Questor, trabalhando com C#, TypeScript e .NET, utilizando Entity Framework e PostgreSQL. Além disso, tenho experiência em projetos para Webscraping com PHP, Laravel, MongoDB e Docker. 

Hoje trabalho com TS, focado em Nest.JS na Epicora Software House, desenvolvendo soluções inteligentes e personalizadas para os mais diversos ramos.

Também desenvolvi projetos buscando soluções personalizadas para empresas de ramos gerais, em Typescript, utilizando o framework do Nest.JS. 

Sou focado em práticas de otimização de código e refatoração, sempre visando a melhoria de performance das aplicações, aliadas a metodologias ágeis. Sou motivado por desafios complexos e busco constantemente criar soluções eficientes que agreguem valor aos negócios.
